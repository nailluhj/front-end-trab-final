HTML

Bem-vindo ao Curso de HTML: Construção de Páginas Web! Neste curso, você mergulhará no mundo da criação de páginas web, aprendendo as habilidades essenciais para criar estruturas de conteúdo sólidas e bem formatadas com HTML. Ao longo deste curso, você estará preparado para construir páginas web profissionais e acessíveis.

Módulo 1: Introdução ao HTML

  • O que é HTML e seu papel na web.
  • Estrutura básica de um documento HTML.
  • Uso de tags HTML para criar elementos e conteúdo.

Módulo 2: Estruturação de Conteúdo

  • Cabeçalhos, parágrafos e quebras de linha.
  • Listas ordenadas e não ordenadas.
  • Elementos semânticos: article, section, header, footer, etc.

Módulo 3: Links e Multimídia

  • Criação de links internos e externos.
  • Incorporação de imagens e vídeos.
  • Uso de áudio em páginas web.

Módulo 4: Formulários e Entrada de Dados

  • Criação de formulários HTML para coleta de informações.
  • Campos de texto, caixas de seleção e botões de envio.
  • Validação de formulários e envio de dados.

Módulo 5: Tabelas e Layout

  • Criação de tabelas para exibição de dados tabulares.
  • Controle de layout com elementos div e span.
  • Uso de CSS para estilização de layout.

Módulo 6: Semântica e Acessibilidade

  • Utilização de elementos semânticos para melhorar a estrutura do conteúdo.
  • Acessibilidade na web: criação de páginas acessíveis.
  • Testes e validação de acessibilidade.

Módulo 7: Melhores Práticas e Publicação

  • Boas práticas de codificação HTML.
  • Hospedagem e publicação de páginas web.
  • Otimização de desempenho e SEO básico.

Módulo 8: Projeto Prático de HTML

  • Aplicação de todas as habilidades aprendidas em um projeto prático.
  • Construção de um site web completo.
  • Feedback e revisão do projeto para melhorias finais.
