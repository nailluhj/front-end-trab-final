JavaScript

Bem-vindo ao Curso de JavaScript: Programação Interativa para a Web! Neste curso, você mergulhará no mundo da programação web, aprendendo as habilidades essenciais para criar interações dinâmicas e funcionais em páginas da web usando JavaScript. Ao longo deste curso, você estará preparado para desenvolver aplicativos web interativos e dinâmicos.

Módulo 1: Introdução ao JavaScript

  • O que é JavaScript e seu papel na web.
  • Sintaxe básica do JavaScript: variáveis, estruturas de controle e funções.
  • Incorporação de JavaScript em páginas web.

Módulo 2: Manipulação do Documento (DOM)

  • Acesso e modificação do DOM com JavaScript.
  • Eventos e manipulação de elementos HTML.
  • Criação de interações do usuário.

Módulo 3: Funções e Objetos em JavaScript

  • Definição e chamada de funções.
  • Objetos em JavaScript: propriedades e métodos.
  • Uso de objetos predefinidos, como Math e Date.

Módulo 4: Manipulação de Dados

  • Trabalhando com arrays e objetos JSON.
  • Laços de repetição: for, while e forEach.
  • Armazenamento local e cookies.

Módulo 5: AJAX e Requisições Assíncronas

  • Requisições HTTP com o objeto XMLHttpRequest.
  • Consumo de APIs web.
  • Tratamento de respostas assíncronas.

Módulo 6:Frameworks e Bibliotecas JavaScript

  • Introdução a bibliotecas populares, como jQuery e React.
  • Uso de frameworks para desenvolvimento web mais eficiente.
  • Aplicação de frameworks em projetos práticos.

Módulo 7: Melhores Práticas e Depuração

  • Boas práticas de codificação JavaScript.
  • Ferramentas de desenvolvimento para depuração de código.
  • Otimização de desempenho.

Módulo 8: Projeto Prático de JavaScript

  • Aplicação de todas as habilidades aprendidas em um projeto prático.
  • Desenvolvimento de um aplicativo web interativo.
  • Avaliação e revisão do projeto para melhorias finais.
